Linkly is a powerful **URL tracking tool** that helps marketers, creators, and developers shorten links, create branded redirects, and monitor click-through performance with precision. This repository contains the public documentation, issue tracker, and developer resources for the Linkly platform.

## Overview

Linkly makes it easy to manage, optimise, and analyse every link you share.
With custom domains, advanced redirect rules, and real-time analytics, Linkly helps you increase trust, boost conversions, and gain deeper insight into your audience.

Key features include:

- Branded short links using your own domains
- Detailed click tracking and event-level analytics
- Geo-redirects, device rules, query-parameter forwarding, and rotators
- QR codes, social preview customisation, and podcast links
- API access for automated link creation and integrations
- Zapier, Make.com, IFTTT, and n8n integration capabilities
- A generous **free plan** offering 1,000 clicks/month

For more details, see the full feature list on the website:
